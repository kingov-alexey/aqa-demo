## 📋 О проекте

**AQA-Demo** — это демонстрационный проект по автоматизации тестирования, созданный для показа лучших практик и подходов в тестировании на стеке **TypeScript + Playwright**.

Проект представляет собой готовый фреймворк для автоматизации тестирования с четкой структурой, разделением на уровни и использованием современных практик разработки тестов.

### 🎯 Цель проекта
- Продемонстрировать профессиональный подход к автоматизации тестирования
- Показать правильную структуру тестового фреймворка
- Использовать современные инструменты и практики
- Служить шаблоном для реальных проектов

## 🚀 Ключевые особенности

### ✅ **Page Object Model**
Реализован паттерн Page Object с наследованием от базового класса:
- Инкапсуляция логики страниц
- Переиспользование компонентов
- Легкая поддержка и расширение

### ✅ **Разделение тестов**
Четкое разделение по типам:
- **UI тесты** — проверка отдельных компонентов
- **API тесты** — тестирование backend
- **E2E тесты** — полные пользовательские сценарии

### ✅ **Работа с данными**
- Централизованное хранение тестовых данных
- Генерация случайных данных
- Поддержка разных окружений (dev/staging/prod)

## 🛠️ Стек технологий

| Инструмент | Назначение |
|------------|------------|
| **Playwright** | Фреймворк для автоматизации тестирования |
| **TypeScript** | Язык программирования с типами |
| **Allure** | Система отчетности |
| **DotEnv** | Управление переменными окружения |
| **Faker** | Генерация тестовых данных |

## 📦 Установка и запуск

### Предварительные требования
- Node.js: latest 20.x, 22.x or 24.x.
- npm или yarn
- Git

### Установка

```bash
# Клонировать репозиторий
git clone https://github.com/kingov-alexey/aqa-demo.git

# Перейти в директорию проекта
cd aqa-demo

# Установить зависимости
npm install

# Установить браузеры Playwright
npx playwright install
